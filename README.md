# Gameloft Test Task - Product Carousel with Shopping Cart

## Overview
This is a React + TypeScript project featuring a product carousel with an interactive shopping cart.  
Users can browse products, add them to the cart, and see the total price with discounts for bulk purchases.

## Features
- Product carousel with next/previous navigation
- Responsive layout:
  - Desktop: 3 products per row
  - Tablet (â‰¥768px): 2 products per row
  - Mobile: 1 product per row
- Interactive shopping cart:
  - Add/remove products
  - Bulk discount: 10% off for quantities > 5
  - Cart preview as a dropdown at the top
- Fully styled using Tailwind CSS
- Written in TypeScript with React and Vite

## Tech Stack
- React (functional components & hooks)
- TypeScript
- Vite for fast development
- Tailwind CSS for styling
